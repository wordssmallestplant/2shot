# 2shot
Welcome to High Dosage, an interactive website full of coffee knowledge. Full JS + CSS applied.

**IMPORTANT INFORMATION**
*Please read before proceeding*

> This website is intended to run on gitHub. To run on Windows Browser separately you must remove
"/" from "/images" in the CSS for the cursor, both for it's border and the custom cursor.

> All visual and audio assets are handmade for "Create-A-Latte," - they are free to use w/o credit.
> All code is free to use and download, if I uploaded it incorrectly and you'd like to access it: contact me @ immon3@crk.umn.edu

**INCLUDED FILES**
> 3 image folders
    - images (misc)
    - images2 (farming/history)
    - images3 (game images)
> css folder with 2 stylesheets
    - stylesheet (general)
    - stylegame (createalatte)
> js folder with 2 scripts
    - script (general)
    - gamescript (createalatte)
> 1 audio folder
    - Contains 5 audio files for createalatte
> 6 html documents
    - index +
    - farming
    - history
    - flavors +
    - coffeematch +
    - createalatte +
> Documents marked with + contain JS in html OR in script/gamescript

**FUN INCLUDED SITE FUNCTIONS**
> Create A Latte is a mini-game complete with sounds. It allows you to build your own latte! (This was...very time intensive).
> Your Perfect Latte makes use of arrays to assign you the perfect latte based off of 3 inputs.
> Random Page Selector on the homepage.
> Cursor is a coffee bean.
> Coffee flavor page has links that redirect you to sites with that flavor of coffee.
